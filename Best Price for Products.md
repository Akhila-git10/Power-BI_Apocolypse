# Power-BI_Apocolypse

## 🧾 Overview
This visualization is to analyze the best store to purchase the products.

## 🛠️ Tools & Technologies
  - Power BI Desktop

## 📂 Dataset
  - Source - Apocolypse Food Prep
  - Format - Excel
  - Includes - Store , Product , Price , Date

## 🚀 How to Use
1. Download the 'Apocolypse Food Prep.xlsx' file.
2. Open it in Power BI Desktop.
3. In Excel Workbook option select the excel file.
4. Select Sheet1.
5. To clean the data click on Transform data button.
6. Once the data gets loaded into Power Query clean the data.
    - In Product section remove Milk category as milk cannot be stored for long period.
    - Change the column name of Date to Date_Purchased.
7. Click on CLose & Apply.
8. Now Data Cleaning is done and dashboard needs to be built.
9. To analyse in which store we have less price for each product.
    -  Select Store and Price columns from the Data section in the right hand side pane.
    -  Select Stacked column chart from Visualisation section.
    -  Drag and drop Store column into Legend in visualisation to categorise with color.
10. Now to analyse which product is economical at which store and place.
    - We shall select Store , Price and Product.
    - For this Clustered Column chart will be suitable.
11. To change the Title for the two charts.
    - Select Visualize in General to enable and edit the Title.
12. To view the exact values enable the Data Labels option and choose none.

- By this visualization from First Chart we analyzed that in Costco the Price is cheaper when compared to the other places.
  
- And by the second chart it is analyzed that for 
  - Rice and Canned Vegetables Target is the best place to purchase.
  - For Dried beans and Bottled water Costco is best.
