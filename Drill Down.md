# Drill Down in Power BI

## 🧾 Overview
The usage of Drill Down option with 'Apocalypse Sales' and 'Purchase Tracker' tables.

## 🛠️ Tools & Technologies
  - Power BI Desktop

## 📂 Dataset
  - Source - Drill Down
  - Format - Excel
  - Includes - Apocalypse Sales , Purchase Tracker

## 🚀 How to Use
1. Download the 'Drill Down.xlsx' file.
2. Load in the Power BI desktop.
3. Both the tables 'Apocalypse Sales' and 'Purchase Tracker' are now visible on the right hand side pane Data.
4. Select Store (Y-axis) and Price(X-axis).
5. Select 'Stacked column chart' in visualization pane.
6. Drag and drop Store in legend for more appropriate visualization with colour.
7. To analyze more with store example Target we'll be using Drill Down in Power BI.
8. Drag and drop 'Product' from 'Purchase Tracker' table in Y-axis under 'Store'.
9. Now Drill Down options are enabled at the bottom right corner of the chart.
10. Select Drill Down and if we selct Target it is further visualized with X-axis Price and Y-axis Product of Target.
11. We can have next level hierarchy.
12. We can also have Expand all one level in the hierarchy.
13. For another example select 'Customer'(Y-axis) from 'Apocalypse Sales' table , 'Units Sold'(X-axis).
14. To Drill Down drag and drop 'Order Id' under 'Customer'.
15. Select 'Stacked Bar Chart' from Visualizations.
16. Select Drill Down from bottom right corner.
17. Select one customer and we can now see each 'Order Id' with the respect to the no. of 'Units sold'.
